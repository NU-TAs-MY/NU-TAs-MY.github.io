## Description
This course will include the following topics: review on number systems: binary number systems, number base conversion, octal and hexadecimal, negative numbers, coded number systems; boolean algebra: main operators, analysis and reduction; design and implementation of combinational circuits: adders, subtractors, decoders, encoders, multiplexers/demultiplexers; sequential circuits: state table and transition diagram, flip-flops; design of sequential circuits: counters, shift registers, memories (ROM, PLA, RAM)â€‹
